# Thinmanj Tap

## How do I install these formulae?

`brew install thinmanj/tap/<formula>`

Or `brew tap thinmanj/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "thinmanj/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
